## 📌 Overview

**Estimate Delivery & Pincode Checker** helps WooCommerce stores improve customer confidence by allowing shoppers to instantly verify:

✅ Delivery availability for their pincode
✅ Estimated delivery date
✅ Estimated delivery days
✅ Courier serviceability via Shiprocket API

Perfect for eCommerce stores that want to reduce cart abandonment and improve customer experience.

---

## ✨ Features

### 🚀 Delivery Availability Check

Verify whether a customer's pincode is serviceable before checkout.

### 📅 Estimated Delivery Date

Display expected delivery dates based on Shiprocket courier data.

### ⏱ Estimated Delivery Days

Show delivery timelines such as:

* Delivery in 2 Days
* Delivery in 4 Days
* Delivery in 7 Days

### 🔗 Shiprocket API Integration

Real-time serviceability and delivery information.

### 🛒 WooCommerce Ready

Works seamlessly with WooCommerce product pages.

### 📱 Responsive Design

Mobile-friendly and optimized for all devices.

### ⚙ Easy Setup

Simple admin configuration panel.

### 📝 Shortcode Support

```shortcode
[delivery_pincode_form]
```

Display the pincode checker anywhere on your website.

## 🔧 Installation

### Method 1

1. Download the plugin ZIP.
2. Go to WordPress Admin → Plugins → Add New.
3. Click Upload Plugin.
4. Upload ZIP file.
5. Activate the plugin.

### Method 2

1. Upload plugin folder to:

```text
/wp-content/plugins/
```

2. Activate plugin from WordPress Admin.

---

## ⚙ Configuration

Navigate to:

```text
WordPress Admin
→ Estimate Delivery Checker
→ Settings
```

Add:

* Shiprocket Email
* Shiprocket Password
* Pickup Pincode

Save settings.
