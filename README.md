Check delivery availability and estimated delivery dates using Shiprocket API.

== Description ==
Estimate Delivery & Pincode Checker allows your customers to check if a pincode is serviceable and when their package might arrive — all powered by Shiprocket API. Easy setup and shortcode support.

== Installation ==
1. Upload the plugin folder to `/wp-content/plugins/`
2. Activate the plugin
3. Configure API credentials in the plugin settings
4. Use `[delivery_pincode_form]` shortcode

== Frequently Asked Questions ==
= Can I show estimated delivery days instead of a fixed date? =
Yes. The plugin auto-calculates the delivery date and days using API.

= How do I uninstall it? =
Simply delete the plugin from WordPress Admin. It also cleans up saved options and DB tables.
